## Konfigurasi
Buat file .env di root direktori proyek dan tambahkan konfigurasi berikut:
DB_NAME=your_database_name
DB_USER=your_database_user
DB_PASSWORD=your_database_password
DB_HOST=localhost
JWT_SECRET=your_jwt_secret
